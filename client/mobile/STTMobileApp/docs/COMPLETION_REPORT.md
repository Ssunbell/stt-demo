## ⚠️ 알려진 제한사항

### 1. 실시간 스트리밍
**현황**: Expo Audio 사용 (제한적)  
**영향**: 완전한 실시간 스트리밍 미지원  
**해결책**: react-native-live-audio-stream 도입 검토

### 2. 오프라인 모드
**현황**: 미지원  
**영향**: 인터넷 연결 필수  
**해결책**: Phase 2에서 구현 예정

### 3. 배터리 소모
**현황**: 연속 녹음 시 소모 증가  
**영향**: 장시간 사용 시 배터리 문제  
**해결책**: VAD 적용, 샘플링 레이트 최적화
